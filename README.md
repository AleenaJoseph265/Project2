# Enterprise Retail Sales Analysis
This project analyzes retail sales data from 01-01-2021 to 08-03-2023 across multiple shopping malls to evaluate revenue performance, customer trends, and product category insights. The goal is to track 2022 performance against targets and identify key growth drivers.
## Tools Used
* SQL
* Power BI
### Process
* Data collection
* ETL
* Data Cleaning - Handled missing values in fields like avgCompanyPosDuratio using mean imputation and categorical blanks using mode
* Dashboard Development - Built interactive filters by Category and Shopping Mall for dynamic analysis
* KPI Tracking - Monitored Overall Revenue, Units Sold, No. of Customers, and YoY Goal Achievement
* Data Visualization
### Key Highlights from Dashboard
1. Revenue Performance - Achieved €251.51M overall revenue with 299K units sold to 99.46K customers. 2022 beat the sales goal by 0.76%, reaching €115.44M vs €114.56M target.
2. Top Performers - Mall of Istanbul and Kanyon led with €51M each in revenue. Clothing dominates as the #1 category with €114M revenue and 104K units sold.
3. Product Analysis - Clothing, Shoes €67M, and Technology €58M are the top 3 categories by revenue. Cosmetics and Toys show lower performance at €7M and €4M.
4. Trend Analysis - Revenue Trend 2021-2023 YTD shows consistent monthly performance, with 2023 data ending 08-03-2023.
